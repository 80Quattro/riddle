API does not work
=================
There are multiple reasons why the API does not work.

Credentials
~~~~~~~~~~~
Double check the *API key* and *API token*

Check different browsers
~~~~~~~~~~~~~~~~~~~~~~~~
It is possible that your browser doesn't show the response from riddle.com.
