Usage
=====

Create new content element
~~~~~~~~~~~~~~~~~~~~~~~~~~
Using this extension is very simple. Create a new plugin on any page and use the **Content Element Wizard** to select the riddle plugin.

Select a riddle
~~~~~~~~~~~~~~~
