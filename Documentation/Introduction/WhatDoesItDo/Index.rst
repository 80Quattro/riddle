What does it do?
----------------

This extension for TYPO3 CMS is an integration of riddles created on https://www.riddle.com.
By using the official API it is easy to embed the riddles into the TYPO3 website.

**Summary of features**

* Easy usage for editors
* Covered with many unit and functional tests
* Actively maintained
